# 🕵️ CTF03 – Leaked Office Incident

## 📖 Scenario

You are a cybersecurity analyst investigating a potential data leak from a company.

You have been provided with a collection of files recovered from an internal system.  
Your task is to analyze the data and uncover hidden information.

---

## 🎯 Objectives

Find all flags hidden in the dataset.

Each flag has the format:

FLAG{something_here}

---

## 📁 Provided Data

- employees.csv – list of employees
- emails.txt – internal communication
- logs.txt – system logs
- image.jpg – suspicious image file
- backup.zip – archived data

---

## 🧠 Tasks

Try to answer the following:

1. Who is the suspicious employee?
2. What password was used in the system?
3. Is there any hidden data in the image?
4. What is inside the backup file?
5. Can you find any unusual activity in logs?

---

## 💡 Tips

- Use Linux tools (`grep`, `cat`, `strings`, `unzip`)
- Inspect files carefully
- Look for hidden data (steganography)
- Think like an attacker

---

## ⚠️ Rules

- Do not brute-force unnecessarily
- Everything you need is inside the provided files

---

Good luck! 🚀

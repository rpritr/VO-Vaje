# Final Project — Security Governance Pack

## Objective
Design a realistic and defensible security governance package for a selected organization of your choice.

## Deliverables
1. Asset Register (20+ assets)
2. Risk Register (15+ risks)
3. Control Mapping (12+ controls)
4. Policy Pack (min 5 policies)
6. BIA + DR Plan

## Video Presentation

Students must record a short presentation explaining their solution.

**Requirements:**

- Record a **3–5 minute video** using **Loom**: https://www.loom.com/
- Briefly explain:
  - the organization you analyzed
  - identified risks and vulnerabilities
  - proposed security improvements
  - key elements of your Security plan
- Share the **public Loom link** in your GitHub repository or submission.

## Evaluation
- Risk realism
- Control justification
- Practical feasibility
- Governance thinking
- Clarity and structure

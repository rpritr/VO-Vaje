# Organizations' Security — Practical Labs (MSc Cyber Security)

This repository contains two intensive lab sessions (8 contact hours total) and a take-home assignment + project work.

## Learning outcomes
By the end, students will be able to:
- identify organizational cyber risks and propose measures/controls,
- structure information security management (governance → controls → evidence),
- draft essential security policies,
- perform a lightweight audit and a business continuity tabletop exercise.

## How submission works
- Each student works in their own repository
- Deliverables are submitted via adding collaborator of the lecturer.
